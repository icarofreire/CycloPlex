# CycloPlex

## Run

```
$ java -jar cycloplex.jar -dir=<dir>
```

Avaliação por blocos:
```
$ java -jar cycloplex.jar -dir=<dir> -b
```

Avaliação por linhas:
```
$ java -jar cycloplex.jar -dir=<dir> -l
```

## Development

Executar projeto:
```
$ gradle run
```

Compilar projeto:
```
$ gradle build
```

Local do arquivo .jar gerado:
```
$ CycloPlex/app/build/libs/
```
